# Cluster-Management
Receive commands from the network, distribute tasks between servers in the cluster, and monitor resource usage per server.

## Tech Stack
- [Java Programming Language](https://www.java.com/en/)
- [Apache Maven](https://www.apache.org/)
- [JavaFX](https://openjfx.io/)

### Adapted Code
This project contains code that was adapted from the following repositories:
- [PCHWRM Client](https://github.com/rnayabed/PCHWRM_Client)
- [PCHWRM Server](https://github.com/rnayabed/PCHWRM_Server)
